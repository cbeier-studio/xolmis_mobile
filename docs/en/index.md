# Xolmis Mobile User Manual

Welcome to Xolmis Mobile.

Xolmis Mobile is a field data collection app designed for ornithologists, researchers, and naturalists. It provides fast, reliable tools for recording species observations, monitoring nests, documenting specimens, and writing field notes — all optimized for offline use and outdoor conditions.

This help guide introduces the main features of the app and provides step‑by‑step instructions to help you get started.

## What You Can Do with Xolmis Mobile

Xolmis Mobile includes several modules, each designed for a specific type of field data:

- [**Inventories**](inventories.md) — record bird observations using multiple sampling methods  
- [**Nests**](nests.md) — document and monitor breeding activity  
- [**Specimens**](specimens.md) — register biological samples collected in the field  
- [**Field Journal**](field-journal.md) — write free‑form notes with formatting, images, and coordinates  
- [**Statistics**](general-statistics.md) — explore summaries and charts of your data  
- [**Settings**](settings.md) — configure the app to match your workflow  

Each module is independent but follows a consistent interface, making the app intuitive and easy to use.

## First Steps

If this is your first time using Xolmis Mobile, start with the [First Steps](first-steps.md) topic.

## Data Management

Xolmis Mobile provides flexible tools for handling your data:

- **Import** and **export** inventories, nests, specimens, and notes  
- Export in **CSV**, **Excel**, **JSON**, and **KML** formats  
- **Back up** and **restore** your entire dataset  

These tools help you integrate your field data with Xolmis Desktop or other analysis workflows.

## Need Help?

Each module includes its own detailed help topic. Use the navigation menu to explore.

Xolmis Mobile is designed to support your fieldwork from start to finish — fast, reliable, and ready for the outdoors.
