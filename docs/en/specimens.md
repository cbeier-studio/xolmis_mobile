# Manage Specimens

The **Specimens** module allows you to record biological samples collected during fieldwork. Each specimen includes a field number, species, locality, sample type, and optional notes or images. This module is designed to support specimen collection workflows and ensure that samples are properly documented for later processing or integration into Xolmis Desktop.

## Create a New Specimen

1. Open the **Specimens** module from the main menu.  
2. Tap the **+** button in the lower-right corner.  
3. Fill in the required fields:
   - **Field number** (generated automatically using the observer abbreviation)  
   - **Species**  
   - **Locality**  
   - **Sample type** (e.g., feather, tissue, carcass, blood, etc.)  
4. Optionally add:
   - Notes  
   - Collection details  
5. Tap **Save**.

Specimens appear in the **Pending** or **Archived** categories depending on their status.

## Edit a Specimen

To update a specimen:

1. Long-press the specimen in the list.  
2. Tap the **Edit** icon.  
3. Modify the fields as needed.  
4. Save your changes.

Edits are applied immediately.

## Attach Images

You can attach photos to document the specimen or its collection context.

1. Open the specimen.  
2. Tap **+**.  
3. Choose **Camera** or **Gallery**.  
4. Optionally add a description.  
5. Save.

Tap an image to view it in full screen.

## Archive Specimens

Specimens can be organized into two categories:

- **Pending** – newly collected or unprocessed specimens  
- **Archived** – specimens that have been processed or finalized  

To change the status:

1. Open the specimen.  
2. Tap the **Archive** button.

Or you can archive various specimen at once:

1. Select specimens using the checkboxes.
2. Tap the **Archive** icon in the bottom bar.
3. Confirm.

## Delete Specimens

### Delete a single specimen

1. Long-press the specimen.  
2. Tap **Delete**.  
3. Confirm.

### Delete multiple specimens

1. Select specimens using the checkboxes.  
2. Tap the **Delete** (trash) icon.  
3. Confirm.

Deleting a specimen removes all associated images.

## Import Specimens

You can import specimens from JSON files exported by another device or user.

1. Open **Specimens**.  
2. Tap the **⋮** menu in the top-right corner.  
3. Select **Import**.  
4. Choose a JSON file.  
5. Wait for the import confirmation.

Import behavior for existing specimens can be configured in **Settings → Import & Export**.

## Export Specimens

Specimens can be exported in bulk.

### Export all specimens

1. Tap the **⋮** menu in the top-right corner.  
2. Select the export format:
   - **CSV**
   - **Excel**
   - **JSON**

A sharing panel will appear so you can send the file to cloud storage or another device.

### Export selected specimens

1. Select specimens using the checkboxes.  
2. Tap the **Export** icon.  
3. Choose the format.
   - **CSV**
   - **Excel**
   - **JSON**
   - **KML**

## Search and Filter Specimens

Use the search bar to find specimens by:

- Field number  
- Species  
- Locality  

Filters allow you to narrow results by:

- Pending / archived  
- Species  
- Locality  
- Date  
- Observer

## Sort Specimens

Tap the **sort icon** to order specimens by:

- Field number  
- Species  
- Locality  
- Date collected  
- Sample type

Sorting helps keep large specimen collections organized.

---

Managing specimens in Xolmis Mobile ensures that biological samples are properly documented and ready for integration into long-term research databases.

*[CSV]: Comma Separated Values
*[JSON]: JavaScript Object Notation
*[KML]: Keyhole Markeup Language
