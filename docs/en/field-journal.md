# Manage Field Journal Notes

The **Field Journal** module allows you to record free‑form notes during fieldwork. These notes can include observations, habitat descriptions, reminders, or any information that does not belong to inventories, nests, or specimens. Each entry may include text, images, and optional GPS coordinates.

This section explains how to create, edit, delete, import, export, search, filter, and sort journal notes.

## Create a New Note

1. Open the **Field Journal** module.  
2. Tap the **+** button in the lower-right corner.
3. Enter your text in the note editor.  
4. Optionally add:
   - **Images**  
   - **GPS coordinates**    
5. Tap **Save**.

New notes appear in the list ordered by date.

## View and Edit Notes

Tap any note to open and edit it.  
From the note screen, you can:

- Read the full text  
- Edit the content  

To edit a note:

1. Tap the note.  
2. Modify the text or attachments.  
3. Save your changes.

## Delete Notes

### Delete a single note
1. Long-press the note.  
2. Tap **Delete**.  
3. Confirm.

### Delete multiple notes
1. Select notes using the checkboxes.  
2. Tap the **Delete** (trash) icon.  
3. Confirm.

Deleting a note does not remove the associated images.

## Import Notes

You can import journal notes from JSON files exported by another device.

1. Open **Field Journal**.  
2. Tap the **⋮** menu in the top-right corner.  
3. Select **Import**.  
4. Choose a JSON file.  
5. Wait for the import confirmation.

## Export Notes

You can export selected notes at once.

1. Select one or more notes.
1. Tap the **Export** icon.  
2. Select the export format:
   - **Plain text** — simple text without formatting.
   - **Markdown** — simple text with formatting marks.
   - **Word** — formatted document compatible with Microsoft Word.
   - **JSON** — structured file used to import in Xolmis.

A sharing panel will appear so you can send the file to cloud storage or another device.

## Search and Filter Notes

Use the search bar to find notes by its content. You can search for keywords, phrases, or specific terms.

Filters allow you to narrow results by:

- Date range  
- Observer

## Sort Notes

Tap the **sort icon** to order notes by:

- Note
- Creation time
- Last modified time  

Sorting helps organize long field campaigns.

---

Managing Field Journal notes ensures that all contextual information from your fieldwork is preserved and easily accessible.
