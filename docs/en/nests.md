# Manage Nests

The **Nests** module allows you to record and manage bird nests found during fieldwork. Each nest includes basic information such as species, locality, field number, and nest support. From this module, you can create, edit, inactivate, reactivate, delete, import, export, search, filter, and sort nests.

This section focuses on managing nests themselves, without covering revisions or eggs.

## Create a New Nest

1. Open the **Nests** module from the main menu.  
2. Tap the **+** button in the lower-right corner.  
3. Fill in the required fields:
   - **Field number** (generated automatically using the observer abbreviation)  
   - **Species**  
   - **Locality**  
   - **Nest support**  
4. Optionally add additional details.  
5. Tap **Save**.

New nests appear in the **Active** tab.

## View and Edit Nest Details

Long-press any nest in the list to open the menu.  
You can edit nest information at any time:

1. Tap the **Edit** icon.  
2. Modify the fields you want to update.  
3. Save your changes.

Edits are applied immediately.

## Inactivate (Finish) a Nest

When monitoring of a nest is complete, you can mark it as **inactive**.

1. Open the nest.  
2. Tap the **Finish** (flag) icon.  
3. Select the final nest outcome (e.g., success, lost, unknown).  
4. Confirm.

Inactive nests move to the **Inactive** tab.

## Reactivate a Nest

If you need to continue monitoring or correct a mistake, you can reactivate an inactive nest.

1. Long-press the nest in the list.  
2. Select **Reactivate**.

The nest returns to the **Active** tab.

## Delete Nests

You can delete one or multiple nests.

### Delete a single nest

1. Long-press the nest.  
2. Tap **Delete**.  
3. Confirm.

### Delete multiple nests

1. Select nests using the checkboxes.  
2. Tap the **Delete** (trash) icon.  
3. Confirm.

Deleting a nest removes all associated data.

## Import Nests

You can import nests from JSON files exported by another device or user.

1. Open **Nests**.  
2. Tap the **⋮** menu in the top-right corner.  
3. Select **Import**.  
4. Choose a JSON file.  
5. Wait for the import confirmation.

Import behavior for existing nests can be configured in **Settings → Import & Export**.

## Export Nests

Nests can be exported individually or in groups.

### Export one nest

1. Long-press the nest.  
2. Choose the export format:
   - **CSV**
   - **Excel**
   - **JSON**
   - **KML**

### Export multiple nests

1. Select nests using the checkboxes.  
2. Tap the **Export** icon.  
3. Choose the format.

### Export all inactive nests

1. Tap the **⋮** menu.  
2. Select **Export all (JSON)**.

## Search and Filter Nests

Use the search bar to find nests by:

- Field number  
- Species  
- Locality  

Filters allow you to narrow results by:

- Active / inactive  
- Species  
- Locality  
- Date
- Nest fate
- Observer

## Sort Nests

Tap the **sort icon** to order nests by:

- Field number  
- Species  
- Locality  
- Date found  
- Last date
- Nest fate

Sorting helps keep large monitoring datasets organized.

---

Managing nests in Xolmis Mobile provides a streamlined workflow for documenting nest discoveries and maintaining accurate breeding records throughout your fieldwork.

*[CSV]: Comma Separated Values
*[JSON]: JavaScript Object Notation
*[KML]: Keyhole Markeup Language
