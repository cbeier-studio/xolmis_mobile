# Editing Field Journal Notes

The Field Journal editor allows you to create rich, structured notes using text formatting, images, and optional GPS coordinates. This section explains how to use the editor effectively.

## Text Formatting

The note editor supports basic formatting to help structure your field notes.

You can apply:

- **Bold** — highlight important terms  
- **Italic** — emphasize species names or notes  
- **Underline** — draw attention to key phrases or warnings  
- **Strikeout** — mark outdated, corrected, or irrelevant text  
- **Inline code** — display short code snippets, commands, or fixed‑width text  
- **Text and background colors** — visually categorize information or highlight important sections
- **Headings** — organize long notes  
- **Bullet lists** — list observations or tasks  
- **Numbered lists** — record sequences or steps  
- **Checklists** — track completed and pending tasks or observations  
- **Code blocks** — show multi‑line code, structured text, or formatted data  
- **Quote blocks** — highlight citations, comments, or important remarks  
- **Links** — reference external resources, documents, or websites  
- **Horizontal dividers** — separate sections  
- **Paragraph alignment and indentation** — structure text visually for clarity and emphasis

Formatting tools appear above the keyboard while editing.

## Adding Images

Images can be attached to any note to document:

- Habitat  
- Weather conditions  
- Tracks or signs  
- Equipment setup  
- Any relevant field context  

### Add an image

1. Open the note.  
2. Scroll to the image will be placed.  
3. Tap **+** camera button.  
4. Choose:
   - **Camera** to take a new photo  
   - **Gallery** to select an existing image    
5. The image will appear in the text.

### Remove images

- Double tap an image to select it.  
- Use the backspace on keyboard to delete it.

## Adding GPS Coordinates

You can attach the device’s current location to a note.

1. Open the note.  
2. Tap **Add coordinates**.  
3. Confirm the location.

Coordinates appear in the text.

## Editing Existing Content

To modify a note:

1. Open the note.  
2. Change the text, formatting, images, or coordinates.  
3. Save your changes.

All edits are applied immediately.

## Best Practices for Editing Notes

- Use **headings** to separate topics (e.g., “Habitat”, “Behavior”, “Weather”).  
- Add **images** to document important observations.  
- Include **coordinates** for location-specific notes.  
- Use **bold** or *italic* to highlight species names or key findings.  
- Keep notes concise but informative.  

---

The Field Journal editor provides a flexible way to capture detailed, well‑structured field notes that complement your inventories, nests, and specimens.
