# First Steps in Xolmis Mobile

This guide helps you get started with Xolmis Mobile by walking through the essential initial settings and the basic steps to begin collecting field data. Completing these first steps ensures that your inventories, nests, specimens, and notes are created correctly and linked to your observer identity.

## 1. Adjust Basic App Settings

Open [**Settings**](settings.md) from the main menu and review the most important options.

### Observer abbreviation

Before using any module, set your personal identification.

A short code used to:

- Generate inventory IDs  
- Create nest and specimen field numbers  
- Identify who collected each record  

Choose a unique and consistent abbreviation (e.g., *CRS*, *JSM*).

### Species search

Select the country where you are working.  

This determines the species list used in searches.  

You can change this later if you work in multiple regions.

### Inventory settings

- Set default durations for timed and point-count inventories  
- Choose the Mackinnon list size  
- Limit the number of active inventories  
- Enable reminders for missing vegetation or weather data  

## 2. Allow Necessary Permissions

When launching the app for the first time, Xolmis Mobile may request access to:

- **Location** (for POIs and coordinates)  
- **Camera and Photos** (for attaching images)  
- **Notifications** (for timed inventories)

Granting these permissions ensures that all features work as intended.

## 3. Create Your First Inventory

Once your settings are configured, you can begin collecting data.

1. Open the [**Inventories**](inventories.md) module.  
2. Tap the **+** button.  
3. Choose an inventory type (e.g., Qualitative, Timed, Transect, Point Count).  
4. Select a locality and start adding species.  

You can run multiple inventories simultaneously if enabled in settings.

## 4. Explore Other Modules

Xolmis Mobile includes several specialized modules:

### Nests

Record and monitor bird nests, including outcomes and metadata. See [Nests](nests.md).

### Specimens

Document biological samples with field numbers, species, and images. See [Specimens](specimens.md).

### Field Journal

Write free-form notes with formatting, images, and optional coordinates. See [Field Journal](field-journal.md).

Each module has its own workflow, but all share consistent navigation and editing tools.

## 5. Back Up Your Data

After finishing your first field session, create a backup:

- Go to **Settings → Backup**  
- Tap **Create backup**  
- Save the file to cloud storage or send it to yourself  

Regular backups protect your data from loss.

## 6. Export or Share Your Data

When you're ready to analyze or share your work:

- Export inventories, nests, or specimens in **CSV**, **Excel**, **JSON**, or **KML** formats  
- Export notes in **Plain text**, **Markdown**, or **JSON**
- Import your data into **Xolmis Desktop** for long-term storage and analysis  

## You're Ready to Begin

With your observer information set, basic settings configured, and your first inventory created, you're ready to use Xolmis Mobile for real fieldwork. As you explore the app, you’ll find that each module is designed to be fast, intuitive, and optimized for outdoor use.
