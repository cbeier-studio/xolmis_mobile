# Eggs

The **Eggs** section allows you to record individual eggs found in a nest, including measurements and notes. This is useful for detailed reproductive studies, biometric datasets, and long-term monitoring.

## Adding an Egg

1. Open the **Nests** module.  
2. Select a nest.  
3. Go to the **Eggs** tab.  
4. Tap the **+** button.  
5. Enter:
   - **Egg field number**
   - **Species**
   - **Egg shape**
   - **Measurements** (length, width, weight)  
6. Save.

Each egg is stored as an individual record.

## Editing an Egg

To modify an egg entry:

1. Long-press the egg in the list.  
2. Select the **Edit** option.
3. Update the fields.  
4. Save your changes.

Egg data can be edited at any time.

## Adding Images to Eggs

Images help document egg morphology, coloration, or damage.

1. Open the egg record.    
2. Tap **+**.  
3. Choose **Camera** or **Gallery**.  
4. Optionally add a description.  
5. Save.

## When to Use Egg Records

Egg records are especially useful when:

- Measuring clutch size  
- Tracking egg development  
- Documenting egg morphology  
- Recording biometric data for research  
- Monitoring nest success or failure
