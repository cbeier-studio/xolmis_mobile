# Ovos

A seção **Ovos** permite registrar individualmente os ovos encontrados em um ninho, incluindo medidas e anotações. Isso é útil para estudos detalhados de reprodução, conjuntos de dados biométricos e monitoramento de longo prazo.

## Adicionando um Ovo

1. Abra o módulo **Ninhos**.  
2. Selecione um ninho.  
3. Vá até a aba **Ovos**.  
4. Toque no botão **+**.  
5. Preencha:
   - **Número de campo do ovo**
   - **Espécie**
   - **Formato do ovo**
   - **Medidas** (comprimento, largura, peso)  
6. Salve.

Cada ovo é armazenado como um registro individual.

## Editando um Ovo

Para modificar um registro de ovo:

1. Toque e segure o ovo na lista.  
2. Selecione a opção **Editar**.  
3. Atualize os campos.  
4. Salve suas alterações.

Os dados do ovo podem ser editados a qualquer momento.

## Adicionando Imagens aos Ovos

Imagens ajudam a documentar morfologia, coloração ou danos no ovo.

1. Abra o registro do ovo.  
2. Toque em **+**.  
3. Escolha **Câmera** ou **Galeria**.  
4. Opcionalmente, adicione uma descrição.  
5. Salve.

## Quando Usar Registros de Ovos

Registros de ovos são especialmente úteis para:

- Medir tamanho da ninhada  
- Acompanhar o desenvolvimento dos ovos  
- Documentar morfologia dos ovos  
- Registrar dados biométricos para pesquisa  
- Monitorar sucesso ou fracasso do ninho
