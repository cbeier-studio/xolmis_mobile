# Xolmis Desktop

[**Xolmis Desktop**](https://github.com/cbeier-studio/Xolmis) é o centro principal para gerenciar, validar, analisar e arquivar os dados coletados com o **Xolmis Mobile**. Enquanto o aplicativo Mobile é otimizado para **entrada rápida de dados em campo, sem necessidade de internet**, o aplicativo Desktop oferece o conjunto completo de ferramentas necessárias para **armazenamento de longo prazo, curadoria e análise científica**.

!!! note
    Este manual é focado no **Xolmis Mobile**. Para instruções detalhadas sobre o uso do **Xolmis Desktop**, consulte a documentação dedicada ao Desktop.

## Capacidades principais

- **Gestão abrangente de dados** – Armazene, edite, valide e faça curadoria de todos os registros ornitológicos.  
- **Ferramentas avançadas de análise** – Gere estatísticas, relatórios, resumos e visualizações.  
- **Módulos estruturados** – Trabalhe com levantamentos, avistamentos, capturas, ninhos, ovos, espécimes, vegetação, clima e muito mais.  
- **Integridade dos dados** – Métodos padronizados, vocabulários controlados e gazetteers hierárquicos.  
- **Arquivamento de longo prazo** – Mantenha um banco de dados centralizado e estável para projetos de pesquisa.

## Fluxo de trabalho com o Xolmis Mobile

1. **Coletar dados** em campo usando o Xolmis Mobile (inventários, ninhos, espécimes, notas).  
2. **Exportar** os dados do Mobile (CSV, Excel, JSON, KML).  
3. **Importar no Desktop** usando o Assistente de Importação do Mobile.  
4. **Verificar e fazer curadoria** dos registros importados.  
5. **Analisar, relatar e arquivar** os dados no ambiente Desktop.  

Esse fluxo garante que as observações de campo se tornem **validadas, padronizadas e armazenadas permanentemente** em um banco de dados robusto de pesquisa.

## Boas práticas

- **Importar logo após o trabalho de campo**: reduz o risco de perda de dados e mantém o banco Desktop atualizado.  
- **Revisar todos os registros importados**: valide identificação de espécies, localidades, contagens e metadados.  
- **Usar vocabulários controlados**: gazetteer, métodos, pesquisadores e instituições devem estar configurados antes da importação.  
- **Fazer backups regularmente**: bancos de dados Desktop contêm dados de pesquisa de longo prazo e devem ser protegidos.

## Relação com o Xolmis Mobile

- **Levantamentos** – Inventários do Mobile tornam‑se eventos de levantamento no Desktop.  
- **Avistamentos** – Detecções de espécies registradas em campo são integradas ao módulo de avistamentos.  
- **Capturas** – Dados de capturas podem ser expandidos com biometria e histórico de anilhamento.  
- **Ninhos e Ovos** – Dados de monitoramento do Mobile são incorporados ao fluxo completo de biologia reprodutiva.  
- **Espécimes** – Amostras coletadas em campo podem ser vinculadas a preparações laboratoriais e coleções institucionais.  

O Xolmis Desktop atua como o **contraponto analítico e arquivístico** do aplicativo Mobile, garantindo que os dados de campo evoluam para informações científicas estruturadas e de alta qualidade.
