# Boas Práticas para Usar o Xolmis Mobile

O Xolmis Mobile foi desenvolvido para tornar a coleta de dados de campo rápida, confiável e cientificamente consistente. Seguir as boas práticas abaixo ajudará você a coletar dados de maior qualidade, evitar erros e garantir uma integração suave com o Xolmis Desktop.

## Prepare-se Antes de Ir a Campo

### Defina sua abreviação de observador

Seu código de observador é necessário para:

- Gerar IDs de inventário  
- Criar números de campo para ninhos e espécimes  
- Registrar quem coletou cada dado  

Certifique-se de configurá-lo corretamente em **[Configurações](settings.md) → Observador**.

### Escolha o país correto para a busca de espécies

Isso garante que a lista de espécies corresponda à lista oficial da sua região.

### Revise suas configurações de inventário

Ajuste:

- Durações padrão  
- Tamanho da lista de Mackinnon  
- Máximo de inventários simultâneos  
- Opções de lembrete para vegetação e clima  

Essas preferências economizam tempo durante o trabalho de campo.

### Carregue seu dispositivo e libere espaço

Fotos, backups e inventários longos exigem bateria e armazenamento.

## Colete Dados de Forma Eficiente em Campo

### Use o tipo de inventário adequado

Escolha o método que corresponde ao seu protocolo:

- Listas temporizadas para esforço padronizado  
- Listas de Mackinnon para avaliações rápidas  
- Transectos e pontos de escuta para levantamentos quantitativos  
- Métodos de detecção para amostragem por distância  

### Adicione espécies rapidamente usando busca parcial

Digitar fragmentos (ex.: `tro mus`, `trmu`, `tromus`) acelera a seleção de espécies.

### Registre POIs quando relevante

Use POIs para:

- Espécies raras  
- Detecções importantes  
- Associações de habitat  

### Adicione dados de vegetação e clima

Esses detalhes contextuais aumentam muito o valor científico dos inventários.

### Evite reativações desnecessárias

Reative inventários apenas quando realmente precisar continuar a amostragem.

## Mantenha Seus Dados Organizados

### Finalize inventários prontamente

Encerrar inventários no momento correto garante:

- Carimbos de tempo precisos  
- Limites corretos da amostra  
- Classificação adequada das espécies (dentro vs. fora da amostra)

### Use notas para observações incomuns

Comportamento, habitat ou detalhes de detecção podem ser adicionados a:

- Espécies  
- POIs  
- Entradas do diário de campo  

### Use filtros e ferramentas de busca

Filtros ajudam você a encontrar rapidamente:

- Inventários ativos vs. finalizados  
- Ninhos ou espécimes específicos  
- Anotações do diário por palavra-chave  

### Ordene listas conforme seu fluxo de trabalho

Ordene por:

- ID ou número de campo  
- Data e hora  
- Localidade  
- Espécie  
- Tipo de inventário  
- Destino do ninho  
- Tipo de espécime  

## Mantenha a Qualidade dos Dados

### Evite duplicar espécies em listas qualitativas

A maioria dos tipos de inventário não permite duplicatas, mas métodos de detecção permitem.

### Verifique contagens de espécies em métodos quantitativos

Toque no número para corrigir valores altos.

### Revise POIs e notas antes de finalizar

Isso evita metadados incompletos ou incorretos.

### Use lembretes para dados ausentes

Ative lembretes para vegetação e clima para evitar lacunas.

## Gerencie Arquivos e Backups com Segurança

### Faça backup dos seus dados regularmente

Use **[Configurações](settings.md) → Backup → Criar backup** após:

- Cada dia de campo  
- Cada expedição  
- Antes de reinstalar o app  
- Antes de trocar de dispositivo  

### Armazene backups na nuvem

Google Drive, OneDrive, Dropbox ou e-mail são boas opções.

### Restaure com cautela

Restaurar um backup **substitui todos os dados atuais**.

## Exporte e Compartilhe Dados Corretamente

### Escolha o formato de exportação adequado

- **CSV** para planilhas e análises estatísticas  
- **Excel** para tabelas estruturadas (experimental)  
- **JSON** para importar no Xolmis Desktop ou Xolmis Mobile  
- **KML** para mapear POIs, ninhos e espécimes  
- **Texto simples** para notas do diário sem formatação  
- **Markdown** para notas do diário com marcações de formatação  

### Exporte apenas inventários finalizados

Inventários ativos devem ser encerrados antes da exportação.

### Use seleção múltipla para exportações em lote

É mais rápido e garante formatação consistente.

## Integre-se Sem Problemas ao Xolmis Desktop

### Importe os dados do Mobile logo após o trabalho de campo

Isso reduz o risco de perda de dados e mantém seu banco atualizado.

### Verifique os registros importados

Cheque:

- Identificações de espécies  
- Localidades  
- Contagens  
- Notas  
- POIs  

### Use o Desktop para armazenamento e análise de longo prazo

O Mobile é para campo; o Desktop é para:

- Validação  
- Relatórios  
- Arquivamento  
- Análises avançadas  

## Mantenha o Aplicativo Funcionando Bem

### Atualize para a versão mais recente

Novas versões incluem:

- Melhorias de desempenho  
- Correções de bugs  
- Taxonomia atualizada  
- Novos recursos  

### Reinicie o app se ele ficar lento

O Xolmis Mobile limpa arquivos temporários automaticamente ao iniciar.

### Evite manter muitos inventários ativos

Limite inventários simultâneos para evitar confusão e erros de dados.

---

Seguir essas boas práticas ajudará você a coletar dados de campo mais limpos, confiáveis e úteis com o Xolmis Mobile — garantindo que suas observações permaneçam valiosas para pesquisas e análises de longo prazo.
