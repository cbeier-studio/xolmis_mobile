# Licenças de Terceiros

O Xolmis Mobile utiliza diversas bibliotecas open‑source de terceiros para fornecer recursos como armazenamento de dados, gráficos, manipulação de arquivos, localização, permissões e componentes de interface. Cada uma dessas dependências é distribuída sob sua própria licença, e o Xolmis Mobile inclui uma tela dedicada onde você pode visualizar todas as informações de licenças de terceiros.

Este tópico explica como as licenças de terceiros são tratadas e como acessá‑las dentro do aplicativo.

## Por que as Licenças de Terceiros Importam

Bibliotecas open‑source são essenciais para o desenvolvimento móvel moderno. Elas fornecem:

- Funcionalidade confiável e testada  
- Ciclos de desenvolvimento mais rápidos  
- Melhorias mantidas pela comunidade  
- Licenciamento transparente e permissivo  

Para cumprir os requisitos de código aberto, o Xolmis Mobile inclui uma lista completa de licenças para todas as dependências usadas no app.

## Visualizando Licenças de Terceiros no App

Você pode acessar a lista completa de licenças diretamente no aplicativo:

1. Abra o **menu principal**.  
2. Toque em **Configurações**.  
3. Role até a seção **Geral**.  
4. Toque em **Sobre o aplicativo**.  
5. Toque em **Licenças de código aberto**.

Essa tela lista todos os pacotes de terceiros junto com seus respectivos textos de licença.

## Tipos de Licenças Utilizadas

A maioria das dependências no Xolmis Mobile utiliza licenças open‑source permissivas, incluindo:

- **MIT License**  
- **BSD License** (2 cláusulas ou 3 cláusulas)  
- **Apache License 2.0**  
- **Licença do Flutter SDK** (estilo BSD)  

Essas licenças geralmente permitem:

- Uso gratuito  
- Modificação  
- Redistribuição  
- Inclusão em apps comerciais  

desde que a atribuição seja preservada.

## Conformidade com Licenças

O Xolmis Mobile cumpre os requisitos de licenciamento open‑source ao:

- Incluir todos os textos de licença no app  
- Fornecer atribuição para cada dependência  
- Respeitar os termos de redistribuição  
- Utilizar apenas bibliotecas com licenças compatíveis  

Nenhuma modificação é feita nas licenças de terceiros, e todos os avisos de direitos autorais são preservados.

## Atualização das Licenças

Sempre que o Xolmis Mobile atualiza suas dependências:

- Novas licenças são incluídas automaticamente  
- Dependências removidas deixam de aparecer  
- A tela **Licenças** sempre reflete o conjunto atual de pacotes  

Isso garante transparência e conformidade contínua.

## Resumo

A seção de Licenças de Terceiros oferece total visibilidade sobre os componentes open‑source utilizados pelo Xolmis Mobile. Essas bibliotecas permitem recursos essenciais enquanto mantêm uma abordagem transparente e responsável em relação ao licenciamento de software.

Você pode revisar todos os textos de licença a qualquer momento em **Configurações → Sobre o aplicativo → Licenças de código aberto**.
