# Manual do Usuário do Xolmis Mobile

Bem-vindo ao Xolmis Mobile.

O Xolmis Mobile é um aplicativo de coleta de dados de campo desenvolvido para ornitólogos, pesquisadores e naturalistas. Ele oferece ferramentas rápidas e confiáveis para registrar observações de espécies, monitorar ninhos, documentar espécimes e escrever anotações de campo — tudo otimizado para uso offline e em condições externas.

Este guia de ajuda apresenta os principais recursos do aplicativo e fornece instruções passo a passo para você começar a utilizá-lo.

## O que você pode fazer com o Xolmis Mobile

O Xolmis Mobile inclui vários módulos, cada um projetado para um tipo específico de dado de campo:

- [**Inventários**](inventories.md) — registre observações de aves usando múltiplos métodos de amostragem  
- [**Ninhos**](nests.md) — documente e monitore atividade reprodutiva  
- [**Espécimes**](specimens.md) — registre amostras biológicas coletadas em campo  
- [**Diário de Campo**](field-journal.md) — escreva anotações livres com formatação, imagens e coordenadas  
- [**Estatísticas**](general-statistics.md) — explore resumos e gráficos dos seus dados  
- [**Configurações**](settings.md) — ajuste o aplicativo ao seu fluxo de trabalho  

Cada módulo é independente, mas segue uma interface consistente, tornando o app intuitivo e fácil de usar.

## Primeiros Passos

Se esta é sua primeira vez usando o Xolmis Mobile, comece pelo tópico [Primeiros Passos](first-steps.md).

## Gerenciamento de Dados

O Xolmis Mobile oferece ferramentas flexíveis para lidar com seus dados:

- **Importe** e **exporte** inventários, ninhos, espécimes e anotações  
- Exporte em formatos **CSV**, **Excel**, **JSON** e **KML**  
- **Faça backup** e **restaure** todo o seu conjunto de dados  

Essas ferramentas ajudam a integrar seus dados de campo ao Xolmis Desktop ou a outros fluxos de análise.

## Precisa de ajuda?

Cada módulo possui seu próprio tópico detalhado de ajuda. Use o menu de navegação para explorar.

O Xolmis Mobile foi projetado para apoiar seu trabalho de campo do início ao fim — rápido, confiável e pronto para o ambiente externo.
