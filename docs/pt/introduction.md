# Introdução

## Por que Xolmis?

Dados ornitológicos são frequentemente coletados em campo sob condições desafiadoras e inicialmente registrados manualmente em papel. Mais tarde, esses registros são digitalizados para análise, mas mesmo assim costumam ser armazenados em formatos diversos, como arquivos CSV, planilhas Excel ou documentos de texto — cada um com sua própria estrutura e convenções definidas por pesquisadores individuais.

Como resultado, os dados podem ficar espalhados por várias pastas, dispositivos ou até computadores de colaboradores. Localizar informações específicas ou lembrar o que existe pode se tornar um verdadeiro desafio. Reutilizar dados de diferentes coleções ou projetos geralmente exige um trabalho tedioso de **padronização, reformatamento e limpeza**. Embora existam plataformas colaborativas de ciência, uma parte significativa de dados valiosos permanece isolada em máquinas pessoais, inacessível à comunidade científica mais ampla.

O **Xolmis** foi criado para enfrentar essa fragmentação. Ele oferece uma solução unificada para **coletar, organizar, curar e gerenciar dados ornitológicos de diversos tipos**. Projetado para pesquisadores, técnicos de campo, instituições e projetos de conservação, o Xolmis fornece um ambiente centralizado e estruturado que promove:

- **Consistência dos dados** – formatos padronizados entre projetos.  
- **Rastreabilidade** – cada alteração é registrada e vinculada a um usuário.  
- **Reuso** – conjuntos de dados podem ser compartilhados, exportados e integrados a outras plataformas.  
- **Colaboração** – vários usuários podem trabalhar no mesmo banco de dados com papéis e permissões definidos.

## O que é o Xolmis?

Xolmis é um **sistema integrado para coleta e gerenciamento de dados ornitológicos**. Ele consolida conjuntos de dados dispersos (planilhas, anotações de campo e registros digitais) em um banco de dados coerente, pesquisável e seguro.

Seus principais objetivos incluem:

- **Agregar diversos tipos de dados** provenientes de pesquisas e observações de campo (capturas, avistamentos, ninhos, espécimes, permissões, projetos etc.).  
- **Facilitar a curadoria e validação dos dados**, garantindo precisão e confiabilidade.  
- **Relacionar registros entre módulos**, proporcionando insights ecológicos e biológicos mais ricos.  
- **Permitir compartilhamento e preservação de longo prazo**, reduzindo o risco de perda de dados.  
- **Aumentar a consistência, rastreabilidade e segurança** nos fluxos de trabalho de pesquisa ornitológica.

!!! info
    _Xolmis_ é um gênero de passeriformes neotropicais. Atualmente é representado por duas espécies: [_Xolmis irupero_](https://ebird.org/species/whimon1) e [_Xolmis velatus_](https://ebird.org/species/whrmon2). O nome foi escolhido para simbolizar o foco do sistema em ornitologia e sua conexão com a biodiversidade neotropical.

## Aplicações

O Xolmis é composto por duas aplicações distintas, que podem ser usadas de forma independente ou integrada, dependendo do fluxo de trabalho:

- **Xolmis Mobile**:  
  Um companheiro leve para o campo, projetado para **coleta de dados in situ**. Ele permite que pesquisadores registrem capturas, avistamentos e outras observações diretamente em dispositivos móveis, exportando depois em formatos padronizados (CSV ou JSON).

- [**Xolmis Desktop**](xolmis-desktop.md):  
  Uma aplicação completa para **gerenciamento de conjuntos de dados complexos**. Ela oferece ferramentas avançadas para editar, verificar, importar, exportar e analisar dados ornitológicos, além de integração com plataformas externas como o eBird.

Juntas, essas aplicações criam um fluxo contínuo: **coletar no campo → consolidar no banco de dados → analisar e compartilhar resultados**.

## Boas práticas ao começar com o Xolmis

- **Centralize seus dados**: importe planilhas e anotações dispersas para um único banco de dados.  
- **Defina papéis de usuário**: atribua permissões a colaboradores para garantir responsabilidade.  
- **Use módulos padronizados**: registre dados no módulo apropriado (ex.: Capturas, Ninhos, Avistamentos) para manter consistência.  
- **Aproveite a integração Mobile + Desktop**: colete dados em campo com o Xolmis Mobile e gerencie-os depois no Xolmis Desktop.  
- **Planeje a preservação de longo prazo**: configure backups e exporte conjuntos de dados regularmente para evitar perda de informações.

Ao adotar o Xolmis, pesquisadores e instituições ganham uma plataforma **robusta, escalável e colaborativa** para gerenciamento de dados ornitológicos, garantindo que informações valiosas sejam preservadas, acessíveis e prontas para apoiar a conservação e a descoberta científica.
