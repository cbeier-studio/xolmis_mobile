# Configurações

O módulo **Configurações** permite ajustar como o Xolmis Mobile se comporta durante o trabalho de campo. Essas opções controlam a busca de espécies, o comportamento dos inventários, a importação/exportação de dados, backups e preferências gerais. Ajustar as configurações antes de ir a campo garante um fluxo de trabalho mais suave e eficiente.

## Configurações Gerais

Preferências gerais da interface do aplicativo.

### Módulo inicial

Selecione qual módulo é exibido quando o app inicia.

### Aparência

Escolha entre:

- Claro
- Escuro
- Padrão do sistema

### País da busca de espécies

Selecione o país onde você está trabalhando.

Isso determina:

- A lista de espécies usada nas buscas
- Listas regionais

### Tags

Gerencie as tags disponíveis. Você pode adicionar novas tags usando o campo no topo da tela, remover tags adicionadas ou alterar a cor das tags. Algumas tags pré-definidas não podem ser removidas.

## Observadores

As configurações do observador definem sua identificação pessoal dentro do aplicativo.

### Abreviação padrão do observador

Um código curto usado para:

- Gerar IDs de inventário
- Criar números de campo de ninhos e espécimes
- Identificar quem coletou cada registro

Essa abreviação é obrigatória para a maioria dos módulos.

## Configurações de Inventário

Configurações que afetam como os inventários se comportam durante a criação e amostragem.

### Máximo de inventários ativos

Limite de quantos inventários podem estar ativos simultaneamente.

### Tamanho da lista de Mackinnon

Defina o número de espécies por lista.

### Durações padrão

Configure tempos padrão para:

- Listas temporizadas
- Listas por intervalos
- Pontos de escuta

### Lembretes

Ative lembretes para:

- Dados de vegetação ausentes
- Dados de clima ausentes

Esses lembretes aparecem ao finalizar um inventário.

## Importação e Exportação

Configurações gerais para transferência de dados.

### Registros existentes ao importar

Defina o que acontece quando os dados importados conflitam com registros existentes:

- Sempre perguntar
- Atualizar registros existentes
- Ignorar registros existentes

### Formatar números

Marque esta opção para usar o separador decimal local, ou desmarque para usar ponto (.).

## Backup

Ferramentas para proteger seus dados.

### Criar backup

Gera um backup completo de todos os dados do aplicativo.

### Restaurar backup

Substitui todos os dados atuais pelo conteúdo de um arquivo de backup.
