# Dados de Clima em Inventários

As condições climáticas podem influenciar fortemente a detectabilidade das aves, seus padrões de atividade e a qualidade do levantamento. O Xolmis Mobile permite registrar informações climáticas básicas para cada inventário, garantindo que seus dados possam ser interpretados com precisão durante análises posteriores.

O registro de dados climáticos é opcional, mas altamente recomendado para levantamentos padronizados.

## O que os Dados de Clima Incluem

Cada registro climático contém os seguintes campos:

- **Cobertura de nuvens** – condição geral do céu  
- **Precipitação** – chuva, garoa, nenhuma etc.  
- **Temperatura** – temperatura ambiente no momento da amostragem  
- **Velocidade do vento** – estimada ou medida  
- **Direção do vento** – direção cardeal  
- **Pressão atmosférica** – campo opcional  
- **Umidade relativa** – campo opcional  

Esses campos fornecem um retrato das condições ambientais durante o inventário.

## Adicionando Dados de Clima

Para adicionar dados de clima:

1. Abra o inventário.  
2. Vá até a aba **Clima**.  
3. Toque no botão **+** no canto inferior direito.  
4. Selecione **Dados de clima**.  
5. Preencha os campos disponíveis.  
6. Salve o registro.

## Editando Dados de Clima

Para editar um registro climático existente:

1. Toque e segure o registro na lista.  
2. Toque no ícone **Editar**.  
3. Atualize os campos conforme necessário.  
4. Salve suas alterações.

As mudanças são aplicadas imediatamente.

## Lembretes de Clima

O Xolmis Mobile pode lembrar você de preencher os dados de clima ao finalizar um inventário.

Para ativar os lembretes:

- Vá em **Configurações → Inventários**  
- Ative **Lembrar clima ausente**

Ao finalizar um inventário sem dados de clima:

- Um diálogo aparece com opções para:
  - **Cancelar** (voltar ao inventário)  
  - **Ignorar** (finalizar sem dados de clima)  
  - **Adicionar** (abrir o formulário de clima)

Isso ajuda a manter consistência na coleta de dados entre levantamentos.

## Exportando Dados de Clima

As informações climáticas são incluídas ao exportar inventários em:

- **CSV**  
- **Excel**  
- **JSON**

Os dados de clima não são incluídos em exportações KML (que contêm apenas coordenadas geográficas).

## Quando Registrar Dados de Clima

Os dados de clima são especialmente úteis quando:

- Realizando pontos de escuta ou transectos padronizados  
- Comparando levantamentos entre dias ou estações  
- Estudando detectabilidade de espécies sob diferentes condições  
- Documentando o contexto ambiental em monitoramentos de longo prazo  

Mesmo anotações climáticas simples podem melhorar significativamente a interpretação dos seus dados de campo.

---

Os dados de clima fornecem contexto essencial para entender detecções de espécies e ajudam a garantir que seus inventários permaneçam cientificamente robustos e comparáveis.
