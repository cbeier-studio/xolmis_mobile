# Editando Notas do Diário de Campo

O editor do Diário de Campo permite criar notas ricas e estruturadas usando formatação de texto, imagens e coordenadas GPS opcionais. Esta seção explica como usar o editor de forma eficiente.

## Formatação de Texto

O editor de notas oferece formatação básica para ajudar a estruturar suas anotações de campo.

Você pode aplicar:

- **Negrito** — destacar termos importantes  
- **Itálico** — enfatizar nomes de espécies ou observações  
- **Sublinhado** — chamar atenção para frases-chave ou avisos  
- **Tachado** — marcar texto desatualizado, corrigido ou irrelevante  
- **Código inline** — exibir pequenos trechos de código, comandos ou texto monoespaçado  
- **Cores de texto e fundo** — categorizar visualmente informações ou destacar trechos importantes  
- **Títulos (headings)** — organizar notas longas  
- **Listas com marcadores** — listar observações ou tarefas  
- **Listas numeradas** — registrar sequências ou etapas  
- **Listas de verificação (checklists)** — acompanhar tarefas ou observações concluídas e pendentes  
- **Blocos de código** — mostrar código multilinha, texto estruturado ou dados formatados  
- **Blocos de citação** — destacar citações, comentários ou observações importantes  
- **Links** — referenciar recursos externos, documentos ou sites  
- **Divisores horizontais** — separar seções  
- **Alinhamento e indentação de parágrafos** — estruturar o texto visualmente para maior clareza e ênfase

As ferramentas de formatação aparecem acima do teclado durante a edição.

## Adicionando Imagens

Imagens podem ser anexadas a qualquer nota para documentar:

- Habitat  
- Condições climáticas  
- Rastros ou sinais  
- Configuração de equipamentos  
- Qualquer contexto relevante de campo  

### Adicionar uma imagem

1. Abra a nota.  
2. Role até o ponto onde a imagem será inserida.  
3. Toque no botão **+** (câmera).  
4. Escolha:
   - **Câmera** para tirar uma nova foto  
   - **Galeria** para selecionar uma imagem existente  
5. A imagem aparecerá no texto.

### Remover imagens

1. Toque duas vezes na imagem para selecioná-la.  
2. Use a tecla de apagar do teclado para removê-la.

## Adicionando Coordenadas GPS

Você pode anexar a localização atual do dispositivo a uma nota.

1. Abra a nota.  
2. Toque em **Adicionar coordenadas**.  
3. Confirme a localização.

As coordenadas aparecerão no texto.

## Adicionando Tags

### Adicionando uma tag

1. Digite um tópico no campo de **Tags**.
2. Escolha uma sugestão da lista ou toque no ícone **+** para adicionar uma nova tag.
3. As tags adicionadas aparecem acima do campo Tags.

### Removendo uma tag

- Toque no **X** da tags para removê-la.

## Editando Conteúdo Existente

Para modificar uma nota:

1. Abra a nota.  
2. Altere o texto, formatação, imagens ou coordenadas.  
3. Opcionalmente, altere as tags.
4. Salve suas alterações.

Todas as edições são aplicadas imediatamente.

## Boas Práticas para Editar Notas

- Use **tags** para separar tópicos (ex.: “habitat”, “comportamento”, “clima”).  
- Adicione **imagens** para documentar observações importantes.  
- Inclua **coordenadas** para notas específicas de localização.  
- Use **negrito** ou *itálico* para destacar nomes de espécies ou achados importantes.  
- Mantenha as notas concisas, mas informativas.  

---

O editor do Diário de Campo oferece uma forma flexível de registrar anotações de campo detalhadas e bem estruturadas, complementando seus inventários, ninhos e espécimes.
