# Dados de Vegetação em Inventários

Os dados de vegetação permitem associar características do habitat a cada inventário. Esses descritores ambientais ajudam a interpretar a composição de espécies, comparar locais de amostragem e apoiar análises ecológicas. Preencher os dados de vegetação é opcional, mas altamente recomendado quando a estrutura do habitat é relevante para o seu estudo.

## O que os Dados de Vegetação Representam

As informações de vegetação no Xolmis Mobile seguem o protocolo da **Alianza del Pastizal**.  
Cada registro de vegetação descreve a estrutura do habitat no local de amostragem, dividida em três categorias:

- **Estrato herbáceo**  
- **Estrato arbustivo**  
- **Estrato arbóreo**

Cada categoria contém três campos:

1. **Distribuição** – como a vegetação está espacialmente organizada  
2. **Proporção** – cobertura estimada  
3. **Altura média** – altura típica do estrato vegetal  

Esses campos ajudam a caracterizar o ambiente de forma padronizada.

## Adicionando Dados de Vegetação

Para adicionar dados de vegetação a um inventário:

1. Abra o inventário.  
2. Vá até a aba **Vegetação**.  
3. Toque no botão **+** no canto inferior direito.  
4. Selecione **Dados de vegetação**.  
5. Preencha os campos dos estratos herbáceo, arbustivo e arbóreo.  
6. Salve o registro.

## Editando Dados de Vegetação

Para editar um registro de vegetação existente:

1. Toque e segure o registro na lista.  
2. Toque no ícone **Editar**.  
3. Modifique os campos conforme necessário.  
4. Salve suas alterações.

As edições são aplicadas imediatamente.

## Anexando Imagens

Registros de vegetação permitem anexar imagens, o que pode ser útil para documentar a estrutura do habitat.

Para adicionar imagens:

1. Abra o registro de vegetação.  
2. Toque no botão **+**.  
3. Escolha:
   - **Câmera** para tirar uma nova foto  
   - **Galeria** para selecionar uma imagem existente  
4. Opcionalmente, adicione uma descrição.  
5. Salve.

Você pode tocar em uma imagem para visualizá-la em tela cheia e aplicar zoom.

## Lembretes para Dados de Vegetação Ausentes

Se ativado em **Configurações → Inventários**, o app pode lembrar você de preencher os dados de vegetação ao finalizar um inventário.

Ao tentar finalizar um inventário sem dados de vegetação:

- Um diálogo de lembrete aparece  
- Você pode escolher:
  - **Cancelar** (voltar ao inventário)  
  - **Ignorar** (finalizar sem dados de vegetação)  
  - **Adicionar** (abrir o formulário de vegetação)

Isso ajuda a manter a consistência na coleta de dados.

## Exportando Dados de Vegetação

As informações de vegetação são incluídas ao exportar inventários em:

- **CSV**  
- **Excel**  
- **JSON**

Imagens anexadas aos registros de vegetação são incluídas em **backups**, mas não nos arquivos exportados.

## Quando Usar Dados de Vegetação

Os dados de vegetação são especialmente úteis quando:

- Comparando tipos de habitat  
- Estudando relações entre espécies e habitat  
- Monitorando mudanças ambientais  
- Documentando condições de amostragem em projetos de longo prazo  

Mesmo um único registro de vegetação por inventário pode aumentar muito o valor ecológico do seu conjunto de dados.

---

Os dados de vegetação fornecem contexto essencial para interpretar detecções de espécies e fortalecem a qualidade científica dos seus inventários.
