# Política de Privacidade do Usuário

Estamos comprometidos em proteger suas informações pessoais.  
Esta Política de Privacidade explica como tratamos seus dados ao usar nossas aplicações.

## 1. O que Coletamos

**O Xolmis não coleta nem armazena quaisquer dados pessoais do usuário.**

Todos os dados ornitológicos inseridos no sistema são **gerenciados localmente pelo usuário** e **nunca são transmitidos para servidores externos** ou compartilhados automaticamente.

## 2. Propriedade e Controle dos Dados

Você mantém **total propriedade e controle** sobre os dados que insere no Xolmis.  
O sistema foi projetado para ajudar você a organizar e gerenciar seus registros ornitológicos **de forma segura e privada**.

## 3. Compartilhamento de Dados

**Nenhum dado é compartilhado com terceiros.**

Qualquer compartilhamento de dados é **inteiramente manual** e fica **a critério do usuário**.

## 4. Dados do Dispositivo e de Uso

O Xolmis não coleta automaticamente informações do dispositivo nem dados de uso.

## 5. Segurança

Embora o Xolmis não transmita dados externamente, recomendamos que os usuários sigam **boas práticas de segurança do dispositivo** e **façam backup de dados importantes**.

## 6. Alterações nesta Política

Esta Política de Privacidade pode ser atualizada no futuro caso o Xolmis passe a coletar ou processar dados de usuários.

**Quaisquer mudanças serão claramente comunicadas** dentro do aplicativo ou em nosso site.
