# Gerenciar Espécimes

O módulo **Espécimes** permite registrar amostras biológicas coletadas durante o trabalho de campo. Cada espécime inclui um número de campo, espécie, localidade, tipo de amostra e notas ou imagens opcionais. Esse módulo foi projetado para apoiar fluxos de coleta de espécimes e garantir que as amostras sejam devidamente documentadas para processamento posterior ou integração ao Xolmis Desktop.

## Criar um Novo Espécime

1. Abra o módulo **Espécimes** no menu principal.  
2. Toque no botão **+** no canto inferior direito.  
3. Preencha os campos obrigatórios:
   - **Número de campo** (gerado automaticamente usando a abreviação do observador)  
   - **Espécie**  
   - **Localidade**  
   - **Tipo de amostra** (ex.: pena, tecido, carcaça, sangue etc.)  
4. Opcionalmente, adicione:
   - Notas  
   - Detalhes da coleta  
5. Toque em **Salvar**.

Os espécimes aparecem nas categorias **Pendentes** ou **Arquivados**, dependendo do status.

## Editar um Espécime

Para atualizar um espécime:

1. Toque e segure o espécime na lista.  
2. Toque no ícone **Editar**.  
3. Modifique os campos necessários.  
4. Salve suas alterações.

As edições são aplicadas imediatamente.

## Anexar Imagens

Você pode anexar fotos para documentar o espécime ou o contexto da coleta.

1. Abra o espécime.  
2. Toque em **+**.  
3. Escolha **Câmera** ou **Galeria**.  
4. Opcionalmente, adicione uma descrição.  
5. Salve.

Toque em uma imagem para visualizá-la em tela cheia.

## Arquivar Espécimes

Os espécimes podem ser organizados em duas categorias:

- **Pendentes** – espécimes recém-coletados ou não processados  
- **Arquivados** – espécimes que já foram processados ou finalizados  

Para alterar o status:

1. Abra o espécime.  
2. Toque no botão **Arquivar**.

Ou você pode arquivar vários espécimes de uma vez:

1. Selecione os espécimes usando as caixas de seleção.  
2. Toque no ícone **Arquivar** na barra inferior.  
3. Confirme.

## Excluir Espécimes

### Excluir um único espécime

1. Toque e segure o espécime.  
2. Toque em **Excluir**.  
3. Confirme.

### Excluir vários espécimes

1. Selecione os espécimes usando as caixas de seleção.  
2. Toque no ícone **Excluir** (lixeira).  
3. Confirme.

Excluir um espécime remove todas as imagens associadas.

## Importar Espécimes

Você pode importar espécimes a partir de arquivos JSON exportados por outro dispositivo ou usuário.

1. Abra **Espécimes**.  
2. Toque no menu **⋮** no canto superior direito.  
3. Selecione **Importar**.  
4. Escolha um arquivo JSON.  
5. Aguarde a confirmação da importação.

O comportamento de importação para espécimes existentes pode ser configurado em **Configurações → Importação e Exportação**.

## Exportar Espécimes

Espécimes podem ser exportados em lote.

### Exportar todos os espécimes

1. Toque no menu **⋮** no canto superior direito.  
2. Selecione o formato de exportação:
   - **CSV**
   - **Excel**
   - **JSON**

Um painel de compartilhamento aparecerá para enviar o arquivo para a nuvem ou outro dispositivo.

### Exportar espécimes selecionados

1. Selecione os espécimes usando as caixas de seleção.  
2. Toque no ícone **Exportar**.  
3. Escolha o formato:
   - **CSV**
   - **Excel**
   - **JSON**
   - **KML**

## Buscar e Filtrar Espécimes

Use a barra de busca para encontrar espécimes por:

- Número de campo  
- Espécie  
- Localidade  

Os filtros permitem restringir os resultados por:

- Pendentes / arquivados  
- Espécie  
- Localidade  
- Data  
- Observador  

## Ordenar Espécimes

Toque no **ícone de ordenação** para ordenar os espécimes por:

- Número de campo  
- Espécie  
- Localidade  
- Data de coleta  
- Tipo de amostra  

A ordenação ajuda a manter grandes coleções de espécimes organizadas.
