# Feedback

Agradecemos muito o seu feedback para ajudar a melhorar o Xolmis Mobile! Seja para relatar um bug ou sugerir um novo recurso, você pode enviar suas contribuições diretamente pelo repositório do GitHub usando a seção **Issues**.

## Como Enviar Feedback

1. Acesse a página de *Issues* do Xolmis Mobile no GitHub:  
   https://github.com/cbeier-studio/xolmis_mobile/issues
2. Clique em **New Issue**
3. Escolha um dos modelos disponíveis:
    - **Bug Report** (Relato de Bug)
    - **Feature Request** (Solicitação de Recurso)
4. Preencha o formulário com o máximo de detalhes possível
5. Clique em **Submit new issue**

!!! note

    Para criar issues no GitHub, você precisa estar logado.  
    Se não estiver, ao clicar em **New issue**, o GitHub solicitará que você faça login.  
    Você pode usar uma conta GitHub existente, entrar com sua conta Google ou criar uma nova conta GitHub, se necessário.

Agradecemos suas contribuições e esperamos tornar o Xolmis Mobile ainda melhor com a sua ajuda!
